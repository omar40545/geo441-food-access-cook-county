## Overview
This project analyses spatial disparities in supermarket access across Cook County using GIS. It integrates social vulnerability, poverty indicators, and accessibility metrics to identify underserved communities.

## Objectives
- Measure supermarket coverage within 0.5 miles
- Compare access with SVI and poverty
- Identify spatial inequities

## Tools Used
- ArcGIS Pro
- Spatial Join & Buffer Analysis
- CDC SVI data

## Key Findings
- Southern and western Cook County show limited access
- Strong correlation between vulnerability and food access gaps
- Spatial inequities suggest policy intervention zones
